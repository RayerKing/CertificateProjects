Něco málo o mně

Jmenuji se David a základy webového vývoje jsem získal už na škole. V současnosti si své znalosti rozšiřuji přes FreeCodeCamp, kde se učím moderní HTML, CSS a JavaScript skrze praktické projekty. Baví mě objevovat, jak web funguje, jak spolu jednotlivé technologie komunikují a jak je možné s trochou kódu vytvořit něco užitečného i vizuálně zajímavého.

Nejlépe se učím metodou pokus-omyl. Teorie a dlouhé přednášky nejsou úplně pro mě. Učím se vlastním tempem, zkouším, co funguje, a hledám cesty, jak se posouvat dál. Mám rád poker, fotbal a Plzeň 🍺. Do budoucna bych se rád ponořil i do Unity a Blenderu, protože mě láká herní vývoj a 3D grafika.

Jsem přátelský, trpělivý a rád řeším věci po svém - hlavně s nadšením a s úsměvem 😄.
